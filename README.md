# Heart Smart Health Promotion Grant Proposal

## Project Overview
Heart Smart is a community-based health promotion initiative designed to reduce cardiovascular disease (CVD) in Australia through education, prevention, and lifestyle interventions.

## Project Highlights
- Community health promotion program design
- Grant funding proposal development
- Stakeholder engagement planning
- Budget preparation and resource allocation
- Program evaluation framework
- Evidence-based public health intervention

## Target Population
- Men aged 40–60 years
- Adults aged 75+ years

## Funding Proposal
- Grant Program: Ian Potter Foundation Public Health Research Grant
- Funding Requested: $37,500

## Key Objectives
1. Increase awareness of cardiovascular disease risk factors.
2. Improve physical activity participation.
3. Promote heart-healthy dietary behaviours.
4. Expand access to preventative health services.
